# 资料下载模块任务书

生成日期：2026-05-07  
适用项目：`E:\ZZY_PROJECT\crmeb_java`

## 零、任务书定位

本模块开发人员只负责编码与确认结果，不负责重新定义资料中心、线索系统或通用文件平台。

## 一、模块目标

建设资料中心，实现：

- 技术资料分类展示
- 公开下载
- 留资后下载
- 下载记录管理

## 二、项目现状与复用基线

现有可复用能力：

- 后台附件管理
- 文件上传接口
- 图片上传接口
- 文件访问能力

结论：

- 底层上传与文件存储复用现有附件系统
- 业务资料中心单独建业务表

## 三、负责人边界

负责人负责：

- 资料中心表结构与代码实现
- 下载登记记录
- 后台资料管理页
- 前台资料中心页

负责人不负责：

- CRM 统一线索平台
- 第三方文件系统替换

## 四、前置输入物料

- 资料分类字典
- 公开下载 / 留资下载规则
- 留资字段清单最终版

## 五、数据库设计基线

### 5.1 设计原则

- 系统附件表只负责底层文件存储
- 资料业务属性单独建表
- 下载登记单独建表

### 5.2 建议表说明

| 表名 | 中文名 | 表用途 | 表类型 | 说明 |
|---|---|---|---|---|
| `eb_download_material` | 资料中心主表 | 保存资料业务信息 | 主表 | 对应资料记录 |
| `eb_download_material_record` | 资料下载登记表 | 保存留资下载记录 | 日志表 | 对应下载登记 |

| 字段名 | 中文含义 | 建议类型 | 必填 | 默认值 | 字段说明 |

## 七、API 设计基线

### 7.1 后台接口

| 接口路径 | 中文名称 | 端别 | 方法 | 用途说明 |
|---|---|---|---|---|
| `/admin/download/material/list` | 资料列表 | admin | `GET/POST` | 获取资料列表 |
| `/admin/download/material/save` | 保存资料 | admin | `POST` | 新增资料 |
| `/admin/download/material/update` | 更新资料 | admin | `POST` | 更新资料 |
| `/admin/download/material/info/{id}` | 资料详情 | admin | `GET` | 获取资料详情 |
| `/admin/download/material/record/list` | 下载记录列表 | admin | `GET/POST` | 获取留资下载记录 |

### 7.2 前台接口

| 接口路径 | 中文名称 | 端别 | 方法 | 用途说明 |
|---|---|---|---|---|
| `/api/front/download/material/list` | 资料列表 | front | `GET` | 获取前台资料列表 |
| `/api/front/download/material/info/{id}` | 资料详情 | front | `GET` | 获取单条资料详情 |
| `/api/front/download/material/record` | 下载登记 | front | `POST` | 留资登记并记录 |
| `/api/front/download/material/file/{id}` | 获取下载文件 | front | `GET` | 返回可下载文件地址 |
