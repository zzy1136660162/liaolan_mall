# 关于辽缆模块任务书

生成日期：2026-05-07  
适用项目：`E:\ZZY_PROJECT\crmeb_java`

## 零、任务书定位

本模块开发人员只负责编码和结果确认，不负责重新设计品牌叙事和官网视觉体系。

## 一、模块目标

建设“关于辽缆”品牌专区，承载：

- 公司介绍
- 发展历程
- 资质荣誉
- 生产实力
- 企业文化
- 联系方式

## 二、项目现状与复用基线

可复用能力：

- 富文本能力
- 图片上传能力
- 页面内容基础能力

结论：

- 以前台固定结构专题页承载
- 后台使用轻量内容块维护

## 三、负责人边界

负责人负责：

- 关于页内容块数据结构
- 前后台页面实现
- 电话与地图入口接入

负责人不负责：

- 整站官网重构
- 品牌文案撰写

## 四、前置输入物料

- 关于页板块顺序最终版
- 公司基础信息最终版
- 地图跳转链接规则

## 五、数据库设计基线

### 5.1 设计原则

- 不做复杂 CMS
- 使用轻量内容块模型
- 板块顺序固定

### 5.2 建议表说明

| 表名 | 中文名 | 表用途 | 表类型 | 说明 |
|---|---|---|---|---|
| `eb_about_page_block` | 关于页内容块表 | 维护关于页各个板块内容 | 主表 | 前台固定结构读取 |

### 5.3 字段说明

#### 表：`eb_about_page_block`

| 字段名 | 中文含义 | 建议类型 | 必填 | 默认值 | 字段说明 |
|---|---|---|---|---|---|
| `id` | 主键 ID | `bigint` | 是 | 自增 | 主键 |
| `block_key` | 板块标识 | `varchar(64)` | 是 | 无 | 固定值，如 `company_intro` |
| `block_name` | 板块名称 | `varchar(128)` | 是 | 无 | 中文板块名称 |
| `title` | 主标题 | `varchar(255)` | 否 | 空字符串 | 板块主标题 |
| `sub_title` | 副标题 | `varchar(255)` | 否 | 空字符串 | 板块副标题 |
| `content` | 内容正文 | `text` | 否 | 空 | 富文本或结构化文本 |
| `image_urls` | 图片集合 | `text` | 否 | 空 | 建议存 JSON 数组 |
| `sort` | 排序值 | `int` | 否 | 0 | 板块排序 |
| `status` | 状态 | `tinyint` | 是 | 1 | 1=启用，0=停用 |
| `created_at` | 创建时间 | `datetime` | 是 | 当前时间 | 创建时间 |
| `updated_at` | 更新时间 | `datetime` | 是 | 当前时间 | 更新时间 |

固定 `block_key`：

- `company_intro`
- `development_history`
- `qualification_honor`
- `production_strength`
- `company_culture`
- `contact_info`

## 六、UI 一致性要求

前台板块顺序固定为：

1. 公司介绍
2. 发展历程
3. 资质荣誉
4. 生产实力
5. 企业文化
6. 联系方式

要求：

- 不允许开发人员调整板块顺序
- 不允许自行修改站点主题色
- 不允许把页面改成营销落地页风格

后台：

- 使用标准表单录入内容块
- 图片上传继续复用现有能力

## 七、API 设计基线

### 7.1 后台接口

| 接口路径 | 中文名称 | 端别 | 方法 | 用途说明 |
|---|---|---|---|---|
| `/admin/about/block/list` | 内容块列表 | admin | `GET/POST` | 获取关于页内容块 |
| `/admin/about/block/save` | 保存内容块 | admin | `POST` | 新增内容块 |
| `/admin/about/block/update` | 更新内容块 | admin | `POST` | 更新内容块 |
| `/admin/about/block/info/{id}` | 内容块详情 | admin | `GET` | 获取单个内容块 |

### 7.2 前台接口

| 接口路径 | 中文名称 | 端别 | 方法 | 用途说明 |
|---|---|---|---|---|
| `/api/front/about/info` | 关于页信息 | front | `GET` | 获取关于页所有内容块 |

### 7.3 参数说明

| 参数名 | 中文含义 | 类型 | 必填 | 来源 | 默认值 | 说明 |
|---|---|---|---|---|---|---|
| `blockKey` | 板块标识 | `string` | 是 | `body` | 无 | 板块唯一标识 |
| `blockName` | 板块名称 | `string` | 是 | `body` | 无 | 中文名称 |
| `title` | 标题 | `string` | 否 | `body` | 空字符串 | 板块标题 |
| `subTitle` | 副标题 | `string` | 否 | `body` | 空字符串 | 副标题 |
| `content` | 内容 | `string` | 否 | `body` | 空字符串 | 正文 |
| `imageUrls` | 图片集合 | `array` | 否 | `body` | 空数组 | 图片列表 |

### 7.4 响应字段说明

| 字段名 | 中文含义 | 类型 | 必返 | 说明 |
|---|---|---|---|---|
| `blockKey` | 板块标识 | `string` | 是 | 前端识别板块用 |
| `blockName` | 板块名称 | `string` | 是 | 中文名 |
| `title` | 标题 | `string` | 是 | 展示标题 |
| `content` | 内容 | `string` | 是 | 展示内容 |
| `imageUrls` | 图片集合 | `array` | 是 | 展示图片 |
