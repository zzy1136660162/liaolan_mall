# 联系我们模块任务书

生成日期：2026-05-07  
适用项目：`E:\ZZY_PROJECT\crmeb_java`

## 零、任务书定位

本模块开发人员只负责编码和结果确认，不负责重新设计 CRM、工单或商机流转平台。

## 一、模块目标

建设标准咨询入口，实现：

- 在线留言
- 客服入口
- 电话拨打
- 地址导航
- 后台留言状态管理

## 二、项目现状与复用基线

现有可复用能力：

- 用户体系
- 表单页面能力
- 电话 / 客服按钮基础组件

当前缺失：

- 业务留言表
- 后台留言管理页
- 留言状态流转

## 三、负责人边界

负责人负责：

- 留言表和接口
- 后台留言管理页
- 前台联系我们页
- 咨询来源埋点

负责人不负责：

- 在线 IM 即时通讯系统
- CRM 商机系统

## 四、前置输入物料

- 留言字段最终版
- 客服二维码 / 链接
- 公司地址与地图链接

## 五、数据库设计基线

### 5.1 设计原则

- 留言独立建业务表
- 来源字段必须保留
- 状态枚举固定

### 5.2 建议表说明

| 表名 | 中文名 | 表用途 | 表类型 | 说明 |
|---|---|---|---|---|
| `eb_contact_message` | 联系我们留言表 | 保存咨询留言 | 主表 | 后台按状态管理 |

### 5.3 字段说明

#### 表：`eb_contact_message`

| 字段名 | 中文含义 | 建议类型 | 必填 | 默认值 | 字段说明 |
|---|---|---|---|---|---|
| `id` | 主键 ID | `bigint` | 是 | 自增 | 主键 |
| `name` | 姓名 | `varchar(128)` | 否 | 空字符串 | 留言人姓名 |
| `phone` | 手机号 | `varchar(32)` | 是 | 无 | 联系电话 |
| `company_name` | 公司名称 | `varchar(255)` | 否 | 空字符串 | 公司名称 |
| `content` | 咨询内容 | `varchar(1000)` | 否 | 空字符串 | 留言内容 |
| `source_page` | 来源页面 | `varchar(64)` | 是 | `contact_page` | 来源标识 |
| `status` | 处理状态 | `varchar(32)` | 是 | `pending` | 留言状态 |
| `handle_remark` | 处理备注 | `varchar(500)` | 否 | 空字符串 | 后台处理说明 |
| `handle_user_id` | 处理人 ID | `bigint` | 否 | 空 | 后台处理管理员 ID |
| `handle_time` | 处理时间 | `datetime` | 否 | 空 | 处理完成时间 |
| `created_at` | 创建时间 | `datetime` | 是 | 当前时间 | 创建时间 |
| `updated_at` | 更新时间 | `datetime` | 是 | 当前时间 | 更新时间 |

固定状态枚举：

- `pending`：待处理
- `contacted`：已联系
- `converted`：已转商机
- `invalid`：无效线索

固定来源枚举：

- `contact_page`
- `product_detail`
- `case_detail`
- `download_center`
- `about_page`

## 六、UI 一致性要求

前台：

- 联系我们页字段顺序固定
- 提交按钮复用主按钮风格
- 电话 / 二维码 / 地址块顺序固定

后台：

- 留言列表使用标准表格
- 状态修改使用固定下拉或固定按钮
- 不创建自定义工单流程界面

## 七、API 设计基线

### 7.1 后台接口

| 接口路径 | 中文名称 | 端别 | 方法 | 用途说明 |
|---|---|---|---|---|
| `/admin/contact/message/list` | 留言列表 | admin | `GET/POST` | 获取留言列表 |
| `/admin/contact/message/info/{id}` | 留言详情 | admin | `GET` | 获取单条留言详情 |
| `/admin/contact/message/update/status` | 更新留言状态 | admin | `POST` | 更新留言状态与处理备注 |

### 7.2 前台接口

| 接口路径 | 中文名称 | 端别 | 方法 | 用途说明 |
|---|---|---|---|---|
| `/api/front/contact/message/save` | 提交留言 | front | `POST` | 提交咨询留言 |
| `/api/front/contact/info` | 联系方式信息 | front | `GET` | 获取电话、地址、二维码等信息 |

### 7.3 请求参数说明

#### 接口：`/api/front/contact/message/save`

| 参数名 | 中文含义 | 类型 | 必填 | 来源 | 默认值 | 说明 |
|---|---|---|---|---|---|---|
| `name` | 姓名 | `string` | 否 | `body` | 空字符串 | 留言人姓名 |
| `phone` | 手机号 | `string` | 是 | `body` | 无 | 联系电话 |
| `companyName` | 公司名称 | `string` | 否 | `body` | 空字符串 | 公司名 |
| `content` | 咨询内容 | `string` | 否 | `body` | 空字符串 | 留言内容 |
| `sourcePage` | 来源页面 | `string` | 是 | `body` | `contact_page` | 来源标识 |

#### 接口：`/admin/contact/message/update/status`

| 参数名 | 中文含义 | 类型 | 必填 | 来源 | 默认值 | 说明 |
|---|---|---|---|---|---|---|
| `id` | 留言 ID | `int` | 是 | `body` | 无 | 目标留言主键 |
| `status` | 状态 | `string` | 是 | `body` | 无 | 固定枚举值 |
| `handleRemark` | 处理备注 | `string` | 否 | `body` | 空字符串 | 备注说明 |

| `companyName` | 公司名称 | `string` | 是 | 公司名称 |
| `content` | 咨询内容 | `string` | 是 | 留言内容 |
| `sourcePage` | 来源页面 | `string` | 是 | 来源枚举 |
| `status` | 状态 | `string` | 是 | 处理状态 |
